🧩 O Desafio do Puzzle 1000 BTC

O famoso "Bitcoin Puzzle" é um desafio criptográfico criado em 2015 onde um desenvolvedor distribuiu 1000 BTC em 256 endereços diferentes, cada um correspondendo a uma private key dentro de faixas de bits específicas.

Como Funciona o Puzzle?

Cada endereço do puzzle possui uma private key que está limitada a uma faixa específica de bits. Por exemplo:

  • Puzzle #1: 1 bit (faixa: 1)
  • Puzzle #10: 10 bits (faixa: 512 a 1023)
  • Puzzle #66: 66 bits (já encontrado em 2023)
  • Puzzle #130: 130 bits (ainda não resolvido)

📚 O Que São Private Keys do Bitcoin?

Uma private key (chave privada) é um número secreto de 256 bits que permite gastar bitcoins associados a um endereço específico. É fundamentalmente a prova matemática de propriedade de uma carteira Bitcoin.
