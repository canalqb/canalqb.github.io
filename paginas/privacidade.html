🔒 Política de Privacidade

Como seus dados são tratados neste site educacional

1. Informações coletadas

Este site não coleta dados pessoais. Não há formulários, cadastros ou armazenamento de identificação do usuário.

Anúncios do Google AdSense podem usar cookies para personalização (veja nossa Política de Cookies).

2. Uso de dados

Nenhuma informação é vendida, alugada ou compartilhada com terceiros. O único uso de dados é para fins estatísticos do Google Analytics (opcional, via consentimento).

3. Seus direitos

Você tem o direito de:

  • Acessar, corrigir ou excluir dados (ainda que nenhum seja coletado);
  • Recusar cookies não essenciais;
  • Desativar anúncios personalizados via banner de consentimento.
